如何启动

你可以选择以下方式启动 MCP 服务：

  • 使用 CLI：fastmcp run mcp_server.py
  • 直接运行脚本：python mcp_server.py

默认启动后可通过 http://127.0.0.1:8000/mcp 访问。

客户端集成示例

在支持 MCP 的客户端（如 TRAE）中可以使用如下配置：

{
  "mcpServers": {
    "file_mcp_server": {
      "command": "mcp-proxy",
      "args": ["http://127.0.0.1:8000/mcp"],
      "description": "通过代理连接到 FastMCP 服务"
    }
  }
}
